Logic Operations Guide

  • Conjunction (AND, ∧): True only when both statements are true.
  • Disjunction (OR, ∨): True when at least one statement is true.
  • Negation (NOT, ¬): Reverses the truth value of a statement.
  • Implication (IF...THEN, →): False only when the antecedent (A) is true and the consequent (B) is false.
  • Biconditional (IF AND ONLY IF, ↔): True when both statements have the same truth value (both true or both false).
